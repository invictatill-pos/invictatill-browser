# InvictaTill Browser 2.0.2

InvictaTill Browser 2.0.2 is a major productivity, WFH intelligence, and auto-updater feature release for Windows 10/11 x64.

## Highlights

- **Built-in InvictaTill AI Cloud Integration**: Ships pre-configured with the official InvictaTill AI API endpoint (`https://invictatill.shop/api/v1/chat`) and built-in API authentication key. Zero manual setup required!
- **24-Hour WFH Productivity & Activity Report**: Keeps detailed 24-hour activity summaries, top visited domain breakdowns, pending vs completed task counts, and instant export to AI Chat & Tasks Drawer.
- **Smart Gmail & Email Task Extractor**: Automatically scans incoming emails (Gmail, Outlook, Yahoo) for action items, subjects, and un-replied emails, converting them into actionable workspace tasks.
- **One-Click Update Installation**: Added explicit "Restart & Install Update Now" buttons across Settings, the Update Banner, Release Notes, and Toast Notifications.
- **Automatic Offline Fallback**: If cloud connectivity is lost, Invicta AI seamlessly switches to local extractive intelligence.
