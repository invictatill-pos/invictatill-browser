# InvictaTill Browser 2.1.7

InvictaTill Browser 2.1.7 includes critical hotfixes for WebRTC screen sharing and the tab screen picker UI.

## Bug Fixes
- **Google Meet Screenshare (Workspace Tabs) Hotfix**: Fixed a bug where the picker was assigning the wrong ID to tabs, and correctly scoped the WebRTC audio request to avoid rejecting valid tab streams.

## Download

- Windows installer: `InvictaTill-Browser-Setup-2.1.7-x64.exe`
- Portable edition: `InvictaTill-Browser-Portable-2.1.7-x64.exe`
