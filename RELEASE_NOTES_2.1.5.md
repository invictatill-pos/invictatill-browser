# InvictaTill Browser 2.1.5

InvictaTill Browser 2.1.5 brings critical bug fixes to the new Opera-style Screen Share Picker, Workspace Tabs, and the Cross-Workspace Password Manager.

## Bug Fixes
- **Google Meet Screenshare (Workspace Tabs)**: Fixed an issue where selecting an InvictaTill Tab from the Screen Share Picker was failing to capture the WebContents stream.
- **Workspace Tab Segregation on Restart**: Fixed a visual bug where tabs restored in the background were temporarily appearing in the Default workspace on application startup.
- **Password Manager Validation Error**: Fixed a DOM binding bug that prevented passwords from being saved into the Cross-Workspace Vault.

- Windows installer: `InvictaTill-Browser-Setup-2.1.5-x64.exe`
- Portable edition: `InvictaTill-Browser-Portable-2.1.5-x64.exe`
