# InvictaTill Browser 2.0.4

InvictaTill Browser 2.0.4 brings top titlebar workspace controls, navigation reliability fixes, and clean tab bar layout!

## Highlights

- **Top Titlebar Workspace Container Selector**: Moved the workspace container button (`Default`, `Work`, `Personal`, custom workspaces) and `+ Add Workspace` option directly onto the top titlebar next to the logo (exactly where requested).
- **Restored Clean Tab Bar Layout**: The tab strip now starts cleanly from the far left of the bar as before.
- **Fixed Navigation & Tab Creation**: Resolved navigation handler exception so creating tabs, searching from NTP shortcuts, and entering web addresses function 100% smoothly without errors.
- **1-Click Toolbar Zoom Controls**: Quick Zoom In (`+`), Zoom Out (`-`), and Zoom Reset (`100%`) buttons directly on the navigation toolbar.

- Windows installer: `InvictaTill-Browser-Setup-2.0.4-x64.exe`
- Portable edition: `InvictaTill-Browser-Portable-2.0.4-x64.exe`
