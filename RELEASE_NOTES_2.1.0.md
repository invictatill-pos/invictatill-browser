# InvictaTill Browser 2.1.0

InvictaTill Browser 2.1.0 fixes tab title text rendering and ensures clean session restore across browser restarts without opening extra blank tabs or extra workspaces!

## Highlights

- **Visible Tab Titles**: Restored tab title DOM element rendering and hostname fallback formatting on all web tabs.
- **Exact Clean Session Restore**: Reopening the browser restores ONLY your active pages and tabs per workspace without opening extra blank tabs or extra workspaces.
- **1-Click Pencil ✏️ & Double-Click Workspace Renaming**: Edit and rename any workspace (including `Default`) instantly using the pencil icon `✏️` or double-clicking the workspace tab.
- **Seamless Meeting Screen Share**: Full screen & window capture support for Google Meet, Zoom, MS Teams, Webex, and Discord meetings.
- **Drag-and-Drop Reordering**: Drag and drop workspace tabs or web tabs to reorder their position.
- **Complete Browser Inside Browser**: Independent tab strips, page state, and session partition for each workspace.
