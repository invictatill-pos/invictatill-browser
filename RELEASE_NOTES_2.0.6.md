# InvictaTill Browser 2.0.6

InvictaTill Browser 2.0.6 introduces Workspace Tabs side-by-side management on the top titlebar!

## Highlights

- **Workspace Tabs Strip**: Workspaces now function like tabs! They are displayed side-by-side on the top titlebar right after the `InvictaTill` logo (`🌐 Default`, `🏢 Work`, `🏠 Personal`).
- **Add & Close Workspaces**: Easily add workspaces one-by-one with the `+ Workspace` button and close them one-by-one with the `✕` close button on each workspace tab.
- **Web Tabs Below**: Open web tabs display cleanly below the workspace header for the active workspace.
- **1-Click Toolbar Zoom Controls**: Quick Zoom In (`+`), Zoom Out (`-`), and Zoom Reset (`100%`) buttons directly on the navigation toolbar.
