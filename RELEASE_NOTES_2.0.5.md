# InvictaTill Browser 2.0.5

InvictaTill Browser 2.0.5 fixes the workspace partition ReferenceError, places the Workspace dropdown & Add Workspace option on the top header right after the logo, and restores the tab strip below it!

## Highlights

- **Fixed Session Partition ReferenceError**: Fixed `setupDownloadHandlers` ReferenceError in main process so `newTab` and workspace switching function 100% reliably.
- **Top Header Workspace Container Selector**: Placed the workspace dropdown selector and `+ Add Workspace` option right after the `InvictaTill` logo on the top titlebar.
- **Clean Tab Bar Below**: Open web tabs display cleanly below the workspace header.
- **1-Click Toolbar Zoom Controls**: Zoom In (`+`), Zoom Out (`-`), and Zoom Reset (`100%`) buttons directly on the navigation toolbar.

## Download

- Windows installer: `InvictaTill-Browser-Setup-2.0.5-x64.exe`
- Portable edition: `InvictaTill-Browser-Portable-2.0.5-x64.exe`
