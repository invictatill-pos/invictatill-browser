# InvictaTill Browser 2.1.8

InvictaTill Browser 2.1.8 is a critical hotfix that fully restores the custom Screen Share Picker UI which was inadvertently suppressed in the previous build.

## Bug Fixes
- **Google Meet Screenshare UI Hotfix**: Fixed an internal function namespace conflict that caused the custom tab and screen picker to be completely suppressed when clicking "Share Screen", leading to immediate WebRTC rejection errors.

## Download

- Windows installer: `InvictaTill-Browser-Setup-2.1.8-x64.exe`
- Portable edition: `InvictaTill-Browser-Portable-2.1.8-x64.exe`
