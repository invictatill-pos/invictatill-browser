# InvictaTill Browser 2.0.7

InvictaTill Browser 2.0.7 introduces Complete Browser-in-Browser Workspace Isolation!

## Highlights

- **Complete Browser Inside Browser**: Each workspace acts as a full, independent browser environment with its own set of tabs, active page state, and session partition.
- **Isolated Tab Strips**: Switching workspaces dynamically loads that workspace's dedicated tab strip. Opening or closing tabs in one workspace does not interfere with other workspaces.
- **Top Header Workspace Tabs**: Workspace tabs are rendered side-by-side on the top header right after the `InvictaTill` logo (`🌐 Default`, `🏢 Work`, `🏠 Personal`).
- **1-Click Workspace Add & Close**: Create new workspaces with `+ Workspace` and close custom workspaces one-by-one with `✕` on each tab.
- **1-Click Toolbar Zoom Controls**: Quick Zoom In (`+`), Zoom Out (`-`), and Zoom Reset (`100%`) buttons directly on the navigation toolbar.
