# InvictaTill Browser 2.0.3

InvictaTill Browser 2.0.3 introduces Multi-Login Workspace Containers and 1-Click Toolbar Zoom Controls!

## Highlights

- **Multi-Login Workspace Containers**: Open and switch workspace containers (`Default`, `Work`, `Personal`, custom workspaces) directly from the top bar dropdown. Each workspace runs in its own isolated session partition, allowing you to log into the exact same website (Gmail, Outlook, GitHub, Slack) with completely different accounts simultaneously in separate tabs!
- **1-Click Toolbar Zoom Controls**: Easily adjust page zoom with Zoom In (`+`), Zoom Out (`-`), and Zoom Reset (`100%`) buttons right on the main toolbar.
- **Color-Coded Workspace Tabs**: Visual color tags on each tab indicating session partition ownership.
- **Built-in InvictaTill AI & WFH Report**: Pre-configured cloud Invicta AI integration, 24-hour activity reporting, and Gmail task extraction.
