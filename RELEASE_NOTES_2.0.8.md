# InvictaTill Browser 2.0.8

InvictaTill Browser 2.0.8 brings Meeting Screen Sharing support, Double-Click Workspace Renaming, and Drag-and-Drop Reordering!

## Highlights

- **Seamless Meeting Screen Share**: Enabled full screen capture support for Google Meet, Zoom, MS Teams, Webex, and Discord meetings.
- **Double-Click Workspace Renaming**: Double-click ANY workspace tab (including `Default`) to rename it instantly.
- **Drag-and-Drop Tab & Workspace Reordering**: Drag and drop workspace tabs or web tabs to reorder their position.
- **Complete Browser Inside Browser**: Independent tab strips, page state, and session partition for each workspace.
- **1-Click Toolbar Zoom Controls**: Quick Zoom In (`+`), Zoom Out (`-`), and Zoom Reset (`100%`) buttons directly on the navigation toolbar.
